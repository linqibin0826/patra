  1. 端口层原则
     • 接口参数改为 ProvenanceCode provenanceCode
     • 无需改动接口的返回值（仍可为 String）
     • 实现层会自动适应

  2. 实现层原则
     • 参数改为 ProvenanceCode provenanceCode
     • 与 MyBatis 交互时: .eq("field", provenanceCode.getCode())
     • 与日志交互时: log.debug("provenance={}", provenanceCode.getCode())

  3. Null 安全
     ```java
     String code = provenanceCode != null ? provenanceCode.getCode() : null;
     ```

  4. DO ↔ Domain 转换
     ```java
     // DO → Domain
     domain.setProvenanceCode(ProvenanceCode.of(doEntity.getProvenanceCode()));
     
     // Domain → DO
     doEntity.setProvenanceCode(domain.getProvenanceCode().getCode());
     ```

  5. 创建转换工具（可选）
     ```java
     public class ProvenanceCodeConverter {
         public static ProvenanceCode fromString(String code) {
             return code == null ? null : ProvenanceCode.of(code);
         }
         
         public static String toString(ProvenanceCode code) {
             return code == null ? null : code.getCode();
         }
     }
     ```
