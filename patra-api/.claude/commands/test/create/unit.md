# Create Unit Test

## Context

You are creating a **unit test** for a domain or application layer class in Papertrace.

### Target Class Information

**Class path argument**: $ARGUMENTS

If no argument provided, ask user for the fully qualified class path (e.g., `com.patra.ingest.domain.model.aggregate.PlanAggregate`).

### Project Context

**Testing framework**: JUnit 5 + Mockito + AssertJ

**Read test examples**:
Look for existing test files in the same module to understand patterns:

### Standards

## Your Task

Create a comprehensive unit test following Papertrace conventions.

### Step 1: Analyze Target Class

1. **Read the target class** to understand:
   - Class responsibilities
   - Public methods to test
   - Dependencies (constructor parameters)
   - Business rules and invariants

2. **Determine test strategy**:
   - **Domain layer** (Aggregate, Entity, VO): Pure unit tests, no mocks
   - **App layer** (Orchestrator): Mock dependencies (repositories, ports)

### Step 2: Determine Test File Location

**Convention**:
- Source: `{module}/src/main/java/com/patra/{service}/domain/model/Foo.java`
- Test: `{module}/src/test/java/com/patra/{service}/domain/model/FooTest.java`

**Verify test directory exists**:
!`ls -la $TARGET_MODULE/src/test/java 2>/dev/null || echo "Test directory needs creation"`

### Step 3: Create Test Class

Write a test class following this template:

```java
package com.patra.{service}.{layer}.{package};

import org.junit.jupiter.api.Test;
import org.junit.jupiter.api.DisplayName;
import org.junit.jupiter.api.BeforeEach;
import org.junit.jupiter.api.Nested;
import static org.assertj.core.api.Assertions.*;
import static org.mockito.Mockito.*;

/**
 * Unit tests for {@link TargetClass}.
 *
 * @author {generated by Claude Code}
 * @since 0.1.0
 */
@DisplayName("TargetClass Tests")
class TargetClassTest {

    private TargetClass target;

    // Mocks (if needed for app layer)
    private DependencyPort mockDependency;

    @BeforeEach
    void setUp() {
        // Initialize mocks
        mockDependency = mock(DependencyPort.class);

        // Create test instance
        target = TargetClass.create(...);
    }

    @Nested
    @DisplayName("Creation Tests")
    class CreationTests {

        @Test
        @DisplayName("should create instance with valid parameters")
        void shouldCreateWithValidParams() {
            // Given
            String code = "test-code";

            // When
            TargetClass instance = TargetClass.create(code);

            // Then
            assertThat(instance).isNotNull();
            assertThat(instance.getCode()).isEqualTo(code);
            assertThat(instance.getStatus()).isEqualTo(ExpectedStatus.DRAFT);
        }

        @Test
        @DisplayName("should throw exception when required parameter is null")
        void shouldThrowWhenRequiredParamNull() {
            // When & Then
            assertThatThrownBy(() -> TargetClass.create(null))
                .isInstanceOf(IllegalArgumentException.class)
                .hasMessageContaining("required");
        }
    }

    @Nested
    @DisplayName("State Transition Tests")
    class StateTransitionTests {

        @Test
        @DisplayName("should transition from DRAFT to ACTIVE")
        void shouldTransitionToActive() {
            // Given
            TargetClass instance = TargetClass.create("test");

            // When
            instance.activate();

            // Then
            assertThat(instance.getStatus()).isEqualTo(Status.ACTIVE);
            assertThat(instance.peekDomainEvents()).hasSize(1);
            assertThat(instance.peekDomainEvents().get(0))
                .isInstanceOf(ActivatedEvent.class);
        }

        @Test
        @DisplayName("should throw exception when invalid state transition")
        void shouldThrowWhenInvalidTransition() {
            // Given
            TargetClass instance = TargetClass.create("test");
            instance.activate();

            // When & Then
            assertThatThrownBy(() -> instance.activate())
                .isInstanceOf(IllegalStateException.class)
                .hasMessageContaining("already active");
        }
    }

    @Nested
    @DisplayName("Business Rule Tests")
    class BusinessRuleTests {

        @Test
        @DisplayName("should enforce business rule X")
        void shouldEnforceRuleX() {
            // Given
            TargetClass instance = TargetClass.create("test");

            // When
            instance.performBusinessOperation();

            // Then
            assertThat(instance.getSomeProperty()).satisfies(prop -> {
                assertThat(prop).isNotNull();
                assertThat(prop).meetsBusinessConstraint();
            });
        }
    }
}
```

### Step 4: Test Coverage Requirements

Ensure tests cover:

- ✅ **Happy path**: All public methods with valid inputs
- ✅ **Edge cases**: Boundary values, empty collections, null checks
- ✅ **Error cases**: Invalid inputs, illegal state transitions
- ✅ **Business rules**: Domain invariants, validation rules
- ✅ **Domain events**: Event emission, event content

**Target coverage**: 75%+ line coverage (per patra-parent/pom.xml)

### Step 5: Naming Conventions

Follow these conventions:

- **Test class**: `{ClassName}Test`
- **Test methods**: `should{ExpectedBehavior}When{Condition}`
- **Display names**: Clear, readable descriptions
- **Nested classes**: Group related tests (CreationTests, StateTransitionTests, etc.)

### Step 6: Assertions

Use AssertJ fluent assertions:

```java
// ✅ Good: Fluent, readable
assertThat(result).isNotNull();
assertThat(result.getCode()).isEqualTo("expected");
assertThat(result.getStatus()).isIn(Status.ACTIVE, Status.COMPLETED);

// ❌ Bad: JUnit assertions
assertTrue(result != null);
assertEquals("expected", result.getCode());
```

### Step 7: Mocking (App Layer Only)

If testing **app layer orchestrator**, mock dependencies:

```java
// Mock repository
when(mockRepository.findById(1L)).thenReturn(Optional.of(aggregate));

// Mock external port
when(mockRegistryPort.fetchConfig(code, op)).thenReturn(snapshot);

// Verify interactions
verify(mockRepository).save(any(Aggregate.class));
verify(mockEventPublisher).publish(anyList());
```

**Domain layer**: NO mocks (pure unit tests)

### Step 8: Verify and Run

After creating test:

1. **Compile check**:
   ```bash
   mvn clean compile test-compile -DskipTests -pl :{module}
   ```

2. **Run test**:
   ```bash
   mvn test -pl :{module} -Dtest=TargetClassTest
   ```

3. **Verify output**: All tests should pass ✅
