1. **遇到问题时**：先在对应分类中查找已知问题
2. **解决疑难问题后**：在对应分类中记录问题和解决方案
3. **记录格式**：保持简洁，一行描述问题 + 解决方案
4. **持续更新**：将新的经验和注意事项及时补充进来
