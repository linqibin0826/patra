# Backlog

## Format
- Story ID, Title, Outcome, Acceptance Criteria, Owner, Status.

## Example
- ING-12, Improve outbox relay throughput, p95 publish < 200ms, metrics show sustained 100 msg/s, Dev A, Todo.

## Process
- Architect curates epics; stories decomposed collaboratively.
- Link stories to ADRs and contracts when applicable.
