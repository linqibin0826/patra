# Dev Task Management

Task management directory for Papertrace development.

## Structure

```
dev/
└── active/           # Active development tasks
    └── [task-name]/
        ├── [task-name]-plan.md      # Comprehensive plan
        ├── [task-name]-context.md   # Key files, decisions, dependencies
        └── [task-name]-tasks.md     # Task checklist
```

## Usage

Use `/dev-docs` command to create structured task plans.
